# Universal Crawler Pro v3.6.14 发布说明

发布日期：2026-07-05

## 下载

- Windows 安装包：`UniversalCrawlerPro_Setup_3.6.14.exe`
- SHA256：`DA7D278688F9A55BE13B376763E52E531679ED81D5569C3F725A3062B1C14DE0`

## 适用环境

- Windows 10 / Windows 11 x64
- 首次启动可能需要等待安全软件扫描安装包
- 用户配置、Cookie 和日志默认写入 `%LOCALAPPDATA%\UniversalCrawlerPro`

## 本版重点

- 统一 GUI 与 WebUI 的七页结构：下载队列、正在下载、已完成、失败列表、日志中心、配置中心、工具箱。
- 优化前端刷新链路，减少高频进度、日志和状态事件造成的界面卡顿。
- 修复打包后资源路径问题，Web 静态文件、`/ui-icon` 图标、GUI/QSS 图标统一走运行时资源解析。
- 修复 Bilibili 输入路由，覆盖 BV 号、UP 主 ID、主页、合集、视频链接、分享链接和关键词等入口。
- 优化 MissAV HLS 下载链路，可调用 `N_m3u8DL-RE.exe`，并补齐必要请求头和本地代理转发策略。
- 优化已完成页媒体信息补救，支持通过 `ffprobe` / `ffmpeg` / 播放器元数据回填时长和分辨率。
- 优化日志中心、失败列表、配置中心、下载详情等页面的表格、筛选、换行和卡片布局。
- 调整下载并发、图片下载并发、日志保留、UI 日志显示数量等设置的实际执行链路。

## 随包组件

- `UniversalCrawlerPro.exe`：桌面 GUI。
- `CrawlerWebPortal.exe`：WebUI / FastAPI 入口。
- `ffmpeg.exe`、`ffprobe.exe`：音视频合并与媒体元数据探测。
- `N_m3u8DL-RE.exe`：HLS 下载。
- Playwright Chromium：用于需要浏览器环境的采集流程。

## 注意事项

- 平台可能存在登录态、Cookie、风控或地区网络限制；请在配置中心确认平台认证状态。
- 安装包不包含个人配置和认证文件，升级时会继续使用本机用户数据目录。
- 如果平台页面长期无结果，优先查看日志中心的 Trace ID 和失败列表中的诊断信息。
