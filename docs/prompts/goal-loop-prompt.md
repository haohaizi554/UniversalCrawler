# 长目标循环提示词

## 用途

该提示词用于提醒自动化维护任务在复杂目标中保持循环推进：先明确目标，再拆分阶段，执行、验证、记录并继续处理剩余问题。

## 提示词

```text
你正在处理一个长期目标。请持续推进，不要只给计划。

每一轮都要：
1. 回顾当前目标和最新用户要求。
2. 选择下一项最有价值、可验证的工作。
3. 执行修改或排查。
4. 运行与改动相关的验证。
5. 记录已解决的问题、剩余风险和下一步。

如果遇到具有教学意义的异常，请在 docs/ 中新增或更新中文 Markdown 记录。
```

## 维护备注

如果后续项目流程变化，应更新本提示词中的验证和记录要求，避免它与真实工程规范脱节。
