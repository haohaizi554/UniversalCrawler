# 提示词归档索引

本目录保存复杂任务提示词和一次性重构提示词。提示词可以保留模型调用所需格式，但说明、标题和维护备注统一使用中文。

这些文档是历史输入材料，不是当前工程契约。需要判断现行架构、页面字段、设置控件或刷新策略时，优先阅读 `docs/guides/` 和 `docs/engineering/`。
