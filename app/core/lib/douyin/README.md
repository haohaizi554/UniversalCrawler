# Douyin 底层库说明

`app/core/lib/douyin/` 保留了抖音底层协议、签名、接口与辅助工具，是项目中体量较大的基础库之一。

## 目录用途

- `encrypt/`
  - 与签名、设备参数、请求标识相关的能力。
- `interface/`
  - 具体接口封装，如详情、用户、合集、搜索等。
- `link/`
  - 分享链接与短链解析。
- `extract/`
  - 数据抽取工具。
- `tools/`
  - 参数、会话、清洗、重试、计时等底层辅助模块。

## 与 GUI 的关系

- GUI 侧并不会直接使用整套底层库的全部能力。
- 当前主要由抖音 spider 流程按需调用其中的参数、链接和接口能力。
- TikTok 相关能力仍保留在底层，但尚未完整接入桌面界面。

## 维护建议

- 变更底层参数结构时，优先确认上层 spider 调用点是否兼容。
- 不建议把新的 GUI 业务逻辑直接塞进该目录。
- 该目录体量较大，重构时优先补回归测试再拆分。

## 分支说明

当前分支没有直接修改该目录源码，但补充了相关测试与目录文档，便于后续继续拆分和维护。
